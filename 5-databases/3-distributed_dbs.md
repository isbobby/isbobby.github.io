# Distributed Databases
## [Two-Phase Commits](https://isbobby.github.io/5-databases/5-3-distributed-dbs/1-2pc.html)
Two-Phase Commit (2PC) is a method used by distributed database to provide transaction guarantee, by coordinating transaction across different shards.
